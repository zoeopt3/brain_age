# Overleaf Upload Instructions

## Quick Start

1. Upload `overleaf_project.zip` to Overleaf:
   - Go to https://www.overleaf.com
   - Click "New Project" → "Upload Project"
   - Select `outputs/release/overleaf_project.zip`

2. Set `main.tex` as the root document (Overleaf usually detects this automatically).

3. Click "Recompile" — the document should compile on the first try.

## Compilation

- **Compiler:** pdfLaTeX (default on Overleaf)
- **Bibliography:** BibTeX (natbib + plainnat style)
- Overleaf handles the pdfLaTeX → BibTeX → pdfLaTeX → pdfLaTeX cycle automatically.

## File Structure

```
main.tex              Main document
references.bib        Bibliography (BibTeX)
figures/              All figures (PNG)
  fig2_*.png          QC distribution plots
  fig3_*.png          Feature distribution plots
  fig4_*.png          Model calibration plots
  fig5_*.png          Explainability plots
```

## Figures

All figures are generated by the pipeline and copied into `figures/`. See `figures/README_FIGURES.md` for the mapping from pipeline outputs to figure filenames used in `main.tex`.

## Notes

- The `\tbd{...}` command highlights items that need completion (shown in red).
- To remove TBD markers for the final version, replace results with full-dataset outputs.
- The document uses standard LaTeX packages available on Overleaf (no custom fonts or external dependencies).
